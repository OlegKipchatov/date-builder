# Date Builder

Минимальный MVP калькулятора дат на Next.js + HeroUI.

## Что умеет

- Считать разницу между двумя датами.
- Считать дни до выбранной даты.
- Считать дни с выбранной даты до сегодня.
- Применять быстрые пресеты: сегодня, +7, +30, конец месяца.
- Сохранять историю расчетов локально в браузере.
- Копировать результат для шаринга.

## Технологии

- Next.js (App Router)
- React 19
- HeroUI
- Tailwind CSS

## Локальный запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Тесты
